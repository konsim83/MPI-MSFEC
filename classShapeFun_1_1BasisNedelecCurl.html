Detailed Description

template<int dim>
class ShapeFun::BasisNedelecCurl< dim >

Curl of Nedelec basis on a given cell.

Class implements curls of vectorial Nedelec basis for a given quadrilateral.

Note
The curls of \(H(\mathrm{curl})\) conforming functions are in \(H(\mathrm{div})\). So we need Piola transforms to guarantee \(H(\mathrm{div})\) conformity.
