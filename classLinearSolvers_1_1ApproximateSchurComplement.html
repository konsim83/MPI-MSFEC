Detailed Description

template<typename BlockMatrixType, typename VectorType, typename PreconditionerType>
class LinearSolvers::ApproximateSchurComplement< BlockMatrixType, VectorType, PreconditionerType >

Implements a serial approximate Schur complement.

Implements a serial approximate Schur complement through the use of a preconditioner for the inner inverse matrix, i.e., if we want to solve

\begin{eqnarray} \left( \begin{array}{cc} A & B^T \\ B & C \end{array} \right) \left( \begin{array}{c} \sigma \\ u \end{array} \right) = \left( \begin{array}{c} 0 \\ u \end{array} \right) \end{eqnarray}

and know that \(A\) is invertible then we choose a preconditioner \(P_A\) and define the approximate Schur complement as

\begin{eqnarray} *\tilde S = C - BP_A^{-1}B^T \end{eqnarray}

to solve for \(u\).
