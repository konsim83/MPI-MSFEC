Detailed Description

template<typename MatrixType, typename PreconditionerType>
class LinearSolvers::InverseMatrix< MatrixType, PreconditionerType >

Implements an iterative inverse.

Implement the inverse matrix of a given matrix through its action by a preconditioned CG solver. This class also works with MPI.

Note
The inverse is not constructed explicitly.
