Detailed Description

\(H(\mathrm{grad})\)- \(H(\mathrm{curl})\) multiscale solver with Lagrange-Nedelec pairings.

This class contains a standard solver for the weighted 1-form Laplacian with rough coefficients in \(H(\mathrm{grad})\)- \(H(\mathrm{curl})\) with Lagrange-Nedelec pairings. The solver is MPI parallel and can be used on clusters.
