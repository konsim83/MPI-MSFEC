Detailed Description

Auxiliary variable \(\sigma = A\nabla\times u\) for 2-forms.

Class defines function for the auxiliary variable \(\sigma = A\nabla\times u\) for an \(H(\mathrm{curl})\)- \(H(\mathrm{div})\) problem, i.e., for 2-forms, from the data class.
