Detailed Description

\(H(\mathrm{grad})\) multiscale solver with Lagrange elements.

This class contains a multiscale solver for the weighted 0-form Laplacian with rough coefficients in \(H(\mathrm{grad})\) with Lagrange elements. The solver is MPI parallel and can be used on clusters.
