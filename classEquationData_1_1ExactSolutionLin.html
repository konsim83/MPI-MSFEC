Detailed Description

Class for analytic solution \(u=Ax+b\).

This class contains the actual implementation of an analytical solution read from a parameter file if \(u\) is a vector proxy, i.e., a 1-form or a 2-form. The analytic solution is

\begin{eqnarray} u(x) = Ax+b \end{eqnarray}

for constant matrices \(A\in\mathbb{R}^{3,3}\) and \(b\in\mathbb{R}^{3}\). The coefficient components are read in from a parameter file.

Note
that if the coefficients of the equation are oscillatory so will be the right hand side. This class is hence mostly only used for sanity checking (semantic debugging) of the implementation for vector proxies.
