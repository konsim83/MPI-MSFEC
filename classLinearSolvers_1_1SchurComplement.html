Detailed Description

template<typename BlockMatrixType, typename VectorType, typename PreconditionerType, typename DoFHandlerType = DoFHandler<3>>
class LinearSolvers::SchurComplement< BlockMatrixType, VectorType, PreconditionerType, DoFHandlerType >

Implements a serial Schur complement.

Implements a serial Schur complement through the use of an inner inverse matrix, i.e., if we want to solve

\begin{eqnarray} \left( \begin{array}{cc} A & B^T \\ B & C \end{array} \right) \left( \begin{array}{c} \sigma \\ u \end{array} \right) = \left( \begin{array}{c} 0 \\ u \end{array} \right) \end{eqnarray}

and know that \(A\) is invertible then we first define the inverse and define the Schur complement as

\begin{eqnarray} \tilde S = C - BP_A^{-1}B^T \end{eqnarray}

to solve for \(u\). The inverse must be separately given to the class as an input argument.
