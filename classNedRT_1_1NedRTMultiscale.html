Detailed Description

\(H(\mathrm{curl})\)- \(H(\mathrm{div})\) multiscale solver with Nedelec-Raviart-Thomas pairings.

This class contains a multiscale solver for the weighted 2-form Laplacian with rough coefficients in \(H(\mathrm{curl})\)- \(H(\mathrm{div})\) with Nedelec-Raviart-Thomas pairings. The solver is MPI parallel and can be used on clusters.
