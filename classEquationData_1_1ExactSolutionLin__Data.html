Detailed Description

Data class to construct an analytic solution.

This class contains only data members read from a parameter file to generate equation data for an analytic solution if \(u\) is a vector proxy, i.e., a 1-form or a 2-form. The analytic solution is

\begin{eqnarray} u(x) = Ax+b \end{eqnarray}

for constant matrices \(A\in\mathbb{R}^{3,3}\) and \(b\in\mathbb{R}^{3}\). The coefficient components are read in from a parameter file.
