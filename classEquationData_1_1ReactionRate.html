Detailed Description

Class for zero order terms. Handle with care! Not used by default.

This zero order term that can possibly regularize the weak form if the k-form is not a vector proxy. If it vanishes we have a Darcy problem for \(k=0\) or \(k=3\).

Note
Handle non-zero values with care since then compatibility of elements is in general not guaranteed!
