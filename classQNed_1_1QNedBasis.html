Detailed Description

Class to hold local mutiscale basis in \(H(\mathrm{grad})\)- \(H(\mathrm{curl})\).

This class is the heart of the multiscale computation. It precomputes the basis on a given cell and assembles the data for the global solver. Once a global solution is computed it takes care of defining the local fine scale solution and writes data.
