Detailed Description

template<int dim>
class ShapeFun::BasisQ1Grad< dim >

Gradient of \(Q_1\) basis on given cell.

Class implements curls of vectorial Nedelec basis for a given quadrilateral.

Note
The gradients of \(H(\mathrm{grad})\) conforming functions are in \(H(\mathrm{curl})\). So we need covariant transforms to guarantee \(H(\mathrm{curl})\) conformity.
