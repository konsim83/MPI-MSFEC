Detailed Description

template<int dim>
class ShapeFun::MyMappingQ1< dim >

Implements \(Q_1\)-mappings.

This class implements \(Q_1\)-mappings in a slightly different way than deal.ii since this is more suitable for our use-case and speeds up things significantly. It stores data for both the mapping from the unit cell to the physical cell and back.
